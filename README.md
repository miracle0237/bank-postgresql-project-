# 💳 Mini Banking System (PostgreSQL)

## 📌 Описание
Учебный проект мини-банковской системы на PostgreSQL.

## ⚙️ Реализовано
- Таблицы: клиенты, счета, транзакции
- Переводы между счетами
- Проверка баланса (нельзя уйти в минус)
- Логирование изменений через триггеры
- Функции на PL/pgSQL

## 🚀 Как запустить

1. Создать базу данных:
CREATE DATABASE bank_db;

2. Подключиться:
\c bank_db;

3. Выполнить файлы:
- schema.sql
- data.sql
- functions.sql
- triggers.sql

## ▶️ Пример использования
SELECT transfer_money(1, 2, 100);
